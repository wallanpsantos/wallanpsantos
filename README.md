### 💼 Experience

**CI&T Software S/A** · Software Engineer · *Nov 2024 – Present*
> Investment Portal (Alta Renda). Migration of critical microservices from Java 8 → Java 17, CI/CD modernization with GitHub Actions & ArgoCD, regulatory solutions (CVM), observability via Dynatrace, test pyramid with JUnit + Mockito + WireMock, and AI-assisted development workflows.

**PicPay Bank** · Software Engineer · *Oct 2021 – Jul 2024*
> Unsecured Loan product at scale. Microservices in Java 17 + Kafka on Kubernetes, monolith decomposition using Clean Architecture, AWS cost optimization (FinOps), load testing with k6, BFF patterns for Mobile, New Relic & Dynatrace observability, and technical mentorship / Code Review.

**Itaú Unibanco S.A.** · Systems Engineering Analyst · *Aug 2019 – Sep 2021*
> Insurance & Capitalization sector. Microservices in Java 11 + Spring Boot for digital products (Home, Auto, Pet Insurance), On-Premise → Hybrid Cloud migration, persistence with MySQL, monitoring via Splunk & AppDynamics.
